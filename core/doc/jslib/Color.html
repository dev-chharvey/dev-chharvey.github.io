Class Color

A 256-bit color that can be displayed in a pixel, given three components (red-green-blue). If the components are not given, they each default to 0 (black).

Constructor

new Color(r, g, b)
Param r
optional integer 0 \le r \le 255 defaults to 0; the red component of this color
Param g
optional integer 0 \le r \le 255 defaults to 0; the green component of this color
Param b
optional integer 0 \le r \le 255 defaults to 0; the blue component of this color

Fields

Instance Variables

this.red
the red component of this color. immutable
this.green
the green component of this color. immutable
this.blue
the blue component of this color. immutable
this.hsv_hue
the HSV-space hue of this color, or what color this color is
this.hsv_sat
the HSV-space saturation, or vividness, of this color
this.hsv_val
the HSV-space value, or brightness, of this color
this.hsl_hue
the HSL-space hue of this color (identical to the HSV-space hue)
this.hsl_sat
the HSL-space saturation of this color, or the amount of color this color has
this.hsl_lum
the HSL-space luminosity of this color, or how white or black this color is

Methods

Instance Methods

this.complement()
Description
Returns a new color that is the complement of this color. The complement of a color is the difference between that color and white. The differences are computed for each component in the RGB-space.
this.invert()
Description
Returns a new color that is the inverse of this color. The inverse of a color is that color with a hue rotation of 180 degrees.
this.brighten(p)
Description
Returns a new color that is a brighter version of this color in the HSL-space.
Param p
required number -1 \le p \le 1; the percentage by which to lighten this color
  • p= 1 will return white
  • p=-1 will return black
  • p= 0 will return an identical copy of this color
this.darken(p)
Description
Returns a new color that is a darker version of this color in the HSL-space.
Param p
required number -1 \le p \le 1; the percentage by which to darken this color
  • p= 1 will return black
  • p=-1 will return white
  • p= 0 will return an identical copy of this color
this.toString(space)
Description
Returns a string representation of this color.
Param space
optional string defaults to 'rgb'; the space in which this color exists
  • space === 'rgb' will return 'rgb(this.red, this.green, this.blue)'
  • space === 'hsv' will return 'hsv(this.hsv_hue, this.hsv_sat, this.hsv_val)'
  • space === 'hsl' will return 'hsl(this.hsl_hue, this.hsl_sat, this.hsl_lum)'

Static Methods

Color.newColorHSV(h, s, v)
Description
Returns a new Color object with a given hue, saturation, and value in HSV-space.
Param h
required integer 0 \le h \le 359; the hue in HSV-space
Param s
required number 0 \le s \le 1; the saturation in HSV-space
Param v
required number 0 \le v \le 1; the value (brightness) in HSV-space
Color.newColorHSL(h, s, l)
Description
Returns a new Color object with a given hue, saturation, and luminosity in HSL-space.
Param h
required integer 0 \le h \le 359; the hue in HSL-space
Param s
required number 0 \le s \le 1; the saturation in HSL-space
Param l
required number 0 \le l \le 1; the luminosity in HSL-space
Color.mix(color1, color2, w)
Description
Mixes two colors, with a weight favoring the first color.
Param color1
required Color object
Param color2
required Color object
Param w
optional number 0 \le w \le 1 defaults to 0.5; the weight favoring the first color
  • w=1 will return color1
  • w=0 will return color2
  • w=0.5 will return a perfectly even mix
