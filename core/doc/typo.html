Code Blocks
The pre Element, by itself (without a child code Element),
will be used for blocks of computer code.

function fibonacci(n) {
  if (n === 0 || n === 1) return 1;
  else if (n > 1)         return fibonacci(n-1) + fibonacci(n-2);
  else                    return NaN;
}

Maps

Note: On all dls below, outlines have been added to each dt and dd to show structure.

Though the HTML5 specification allows for multiple dts and/or dds in any group of a dl, we will not be using that convention here. Each group of a dl must have exactly 1 dt and exactly 1 dd. Thus a dl can be easily converted into a JSON object.

The dl below has three groups, each with one dt and one dd (hereafter referred to as components). The first group’s components simply contain plain text. The dt in the second group contains a list with 3 items, and the dd in the third group also contains a list with 3 items. Using an ol or ul inside a dt or dd allows for including multiple components inside a group. The items are not taken as alternatives, but rather they are all applicable.

Tables

tl;dr: Add class .Table to a table Element to make it appear as an encapsulated unit (like a paragraph). In the .Table, use .Rowgroup on thead, tfoot, and tbody Elements to add the borders above and below. Use subclasses .Head, .Foot, and .Body for specific display properties. Use .Row on tr Elements to add odd and even (or head) coloring. Use .Cell on all th and td Elements to add padding and min-height. Use subclass .Hc on th for a heading look (font-family and bold).

Note: On all .Document objects in this section, outlines have been added to show structure.

A document may take the form of a body > main or body >> article. The said document should have a status and be marked up with metadata. Use the .Document class to style the document, and use modifier subclasses to style the document according to its status if it has one. Notice we are separating semantics with presentation: the [data-status] attribute is unrelated to what class is applied to the Element.
