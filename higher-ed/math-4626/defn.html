Real Limits

Mathematical Definition

The limit as x approaches c of ƒ(x) is L iff:
For every ε > 0 there exists a δ > 0 such that if 0 < |x - c| < δ then |ƒ(x) - L| < ε.

Symbolically Represented

limx→c ƒ(x) = L  ⇔  ∀ε>0 ∃δ>0  ∍  (0 < |x - c| < δ) → (|ƒ(x) - L| < ε)

In English

We can make ƒ(x) as close as we want to L as long as we can find an x close enough, but not equal, to c.

Limits at Infinity

Mathematical Definition

The limit as x increases without bound (approaches ∞) of ƒ(x) is L iff:
For every ε > 0 there exists a δ > 0 such that if x > δ then |ƒ(x) - L| < ε.
And ƒ(x) converges to L.

Symbolically Represented

limx→∞ ƒ(x) = L  ⇔  ∀ε>0 ∃δ>0  ∍  (x > δ) → (|ƒ(x) - L| < ε)

In English

We can make ƒ(x) as close as we want to L as long as we can find an x large enough.

Infinite Limits

Mathematical Definition

As x approaches c, ƒ(x) increases without bound (approaches ∞) iff:
For every ε > 0 there exists a δ > 0 such that if 0 < |x - c| < δ then ƒ(x) > ε.
And the limit of ƒ(x) does not exist.

Symbolically Represented

x→c ⇒ ƒ(x)→∞  ⇔  ∀ε>0 ∃δ>0  ∍  (0 < |x - c| < δ) → (ƒ(x) > ε)

In English

We can make ƒ(x) as large as we want as long as we can find an x close enough, but not equal, to c.
