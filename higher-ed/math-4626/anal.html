Analytical Methods

for calculating limits

Site Map

Without using a table of values or the graph of a function, how can we determine the limit of a given function at a given point? There is an analytical procedure for finding the limit of a function, given the function's equation. We are explicitly computing the limit, and this method is more accurate than using a table or a graph.

1. Direct Substitution

The first analytical method you should try is plugging in c for x to find ƒ(c). The direct substitution method, when valid, tells us that ƒ(c) = limx→c ƒ(x). This method may not always work, but if it does, you do not have to proceed to the next steps.

Example:

Let ƒ(x) = cos(πx)⋅x2 and c = 2. Find limx→c ƒ(x).

Solution:

Find ƒ(c) by direct substitution:
limx→c ƒ(x) = ƒ(c) = cos(πc)⋅c2
limx→2 cos(πx)⋅x2 = ƒ(2) = cos(2π)⋅22
limx→2 cos(πx)⋅x2 = 4

2. Factorization

Sometimes direct substitution doesn't yield an answer because the function is undefined at that point. In this case, we try to simplify the function to a form for which we can use Direct Substitution to find the limit.

Example:

Let ƒ(x) = (x2-1)÷(x-1) and c = 1. Find limx→c ƒ(x).

Solution:

If we try the direct substitution method, we obtain:
limx→c ƒ(x)
limx→1 (x2-1)÷(x-1)
(12-1)÷(1-1)
(1-1)÷(1-1)
0÷0, which does not exist.

Thus we try factoring. Observe that (x2-1) can be factored into (x-1)⋅(x+1). Then:
limx→c ƒ(x)     (goal)
limx→1 (x2-1)÷(x-1)     (given)
limx→1 ((x-1)⋅(x+1))÷(x-1)     (factor)
limx→1 (x+1)÷1     (cancel common factors)
limx→1 x+1     (simplify)
1+1     (direct substitution)
2     (solution)

Note here that limx→1 ƒ(x) = 2, but ƒ(2) does not exist. When we use the Factorization method, we are using Direct Substitution on a simplified form of the function, not the function itself. 3. Rationalization

Rationalization uses the same concept as Factorization: we are trying to simplify a given function into a form that allows us to use Direct Substitution to find the limit. We use the Rationalization method on a function that deals with irrational numbers. Rationalization is basically backwards factorization: instead of dividing by a common factor, we are multiplying by a common factor.

Example:

Let ƒ(x) = (√(x2+9)-3)÷x2 and c = 0. Find limx→c ƒ(x).

Solution:

Clearly the direct substitution method does not work, and there are no expressions in the function that we can factor. But let's try to get rid of that radical.
limx→c ƒ(x)     (goal)
limx→0 (√(x2+9)-3)÷x2     (given)
limx→0 ((√(x2+9)-3)÷x2) ⋅ (1÷1)     (multiply by 1)
limx→0 ((√(x2+9)-3)÷x2) ⋅ ((√(x2+9)+3)÷(√(x2+9)+3))     (rewrite 1 as a quotient of radicals)
limx→0 ((√(x2+9)-3)⋅(√(x2+9)+3)) ÷ (x2⋅(√(x2+9)+3))     (multiply the fractions)
limx→0 ((x2+9)-9) ÷ (x2⋅(√(x2+9)+3))     (expand the numerator)
limx→0 (x2) ÷ (x2⋅(√(x2+9)+3))     (simplify the numerator)
limx→0 1 ÷ (1⋅(√(x2+9)+3))     (cancel common factors)
1÷(1⋅√(02+9)+3)     (direct substitution)
1÷(√(9)+3)     (simplify)
1÷(3+3)     (simplify)
1÷6     (solution)

In conlusion, there are three main methods used to analytically compute the limit of a function. No matter how complicated the function's equation, we will always use some combination of factoring, rationalizing, and substituting to find the limit.
