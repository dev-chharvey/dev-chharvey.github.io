Spreadsheet Applications

TinkerPlots and Excel are two applications for analyzing data, but they are very different. I have included both on the same page because they are both data analysis applications.

My blog post about spreadsheets.

TinkerPlots

TinkerPlots is a program published by Key Curriculum Press™, the same company that published Geometer's Sketchpad. With this application students can import data in spreadsheet form and analyze it. They can create graphs of different types and see different views of those graphs.

Rows are cases and Columns are variables (or attributes). So information in the same row all belongs to the same case (e.g., the same person). This leads to many differences in how spread sheets vs statistics programs operate and the kinds of graphs they are designed to create.

— Cliff Konold, TinkerPlots developer
