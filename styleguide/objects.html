The .o-ListBare Object

Note: On all lists below, outlines have been added to each line item to show structure.

Bare list Objects are semantically no different from normal Lists, except that they are displayed without any padding or list styles. Use this Object for any list of non-textual elements, for example, a list of links in a navigation menu.

Every item in the list should be a .o-ListBare__Item Object.

The .o-ListInline Object

Note: On all lists below, outlines have been added to each line item to show structure.

Inline list Objects are semantically no different from normal Lists. Stylistically, though, they are displayed inline and have inline line items. Essentially, they can be used for listing words, spans, labels, badges, and other inline content. Wrap this Object in a block if necessary.

Every item in the list should be a .o-ListInline__Item Object.

The .h-After∗ Helpers

The following helper classes add content to the ::after pseudo-element of an element. .h-After adds the empty string as content, so it is mostly useless, save for some rare cases.

The following elements act like comma-separated list items. Use this for a list of keywords, for example. These might take the form of lis inside a ol, or they may be simply a list of spans.
