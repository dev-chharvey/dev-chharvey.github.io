Set-Builder Notation

“The set of all x such that P(x) is true…”

\left\{x \mid P(x)\right\}

where P(x) is some rule for x.

Interval Notation

“The set of all x between a and b.”

\left[a,b\right] \left(a,b\right) \left[a,b\right) \left(a,b\right]
a\le x\le b a\lt x\lt b a\le x\lt b a\lt x\le b

They look like ordered pairs, but they're not!
