Learning Objectives

Combinatorics

  • Given a situation with multiple outcomes, count the number of possible outcomes using the Fundamental Counting Principle.
  • Decide whether to use permutations or combinations to calculate a number of possible outcomes.
  • Calculate permutations and combinations using a formula.

Binomial Distribution

Normal Distribution

  • Given the mean and standard deviation of a normally distributed data set, determine
    • the z-score of a sample observation, and
    • the data value of a given z-score.
  • Given a normal distribution, determine
    • the proportion of observations that fell between two given values, and
    • probability that a random observation would fall between two given values.
  • Given a normal distribution and sample size, determine the number of observations that fell between two given values.
  • Given a normal distribution, sample size, and number of observations, determine the proportion of the sample set and possible values between which the observations lie.
  • Interpret and apply the total area under a normal curve using the Unitarity Rule and the Empirical Rule.

Regression

  • Determine the regression model (linear, quadratic, exponential, or logarithmic) of a bivariate data set.
  • Use regression to calculate the equation of the best-fit curve.
  • Use regression to interpolate and extrapolate data given an input or an output.
