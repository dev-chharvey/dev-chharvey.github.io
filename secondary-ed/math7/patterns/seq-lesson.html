Lesson 3.6: Sequences

SOL
7.2 The student will describe and represent arithmetic and geometric sequences, using variable expressions.
Timeframe
Week II.8
Time required
5 days
Lesson created
Dec 1, 2011
Lesson updated
Jan 6, 2014

Preparation

Objectives

  • Identify patterns in sequences
  • Identify the common difference in an arithmetic sequence
  • Identify the common ratio in a geometric sequence
  • Find a specified term of a given sequence
  • Construct a sequence based on a rule

Essential Questions

Vocabulary

sequence
an ordered list of numbers
term
a number in a sequence
arithmetic sequence
a sequence in which the difference between two consecutive terms is constant
common difference
the difference between two consecutive terms in an arithmetic sequence
geometric sequence
a sequence in which the ratio between two consecutive terms is constant
common ratio
the ratio between two consecutive terms in a geometric sequence

Lesson Implementation

MONDAY:
- Go over project rubric to Modeling Sequences Project (due FRIDAY). Play "bump" (where students read and "bump" to the next reader).
- Hand out materials (buttons, toothpicks, tiles, etc.) and let students make patterns from them. Here we do not need to do arithmetic/geometric sequences. They are just getting used to making patterns. Patterns can be based on anything: color, shape, number, etc.
- Give out sequence puzzles for them to work on as an exit slip
- no homework

TUESDAY:
- Introduce arithmetic sequences and give definition
- Give blank four-square: definitions/examples/non-examples/formulas, students fill in
- use classwork to:
  - practice finding common difference
  - practice writing variable expressions
  - find the next 3 terms
- exit slip
- homework: Practice B #1-3,7,9; Practice C #1,3,4,6

WEDNESDAY:
- Introduce geometric sequences and give definition
- Give blank four-square: definitions/examples/non-examples/formulas, students fill in
- use classwork to:
  - practice finding common ratio
  - practice writing variable expressions
  - find the next 3 terms
- exit slip
- homework: 6-8 Practice #1-4,7-10,17-20

THURSDAY:
- warmup: practice distinguishing arithmetic/geometric sequences
- notes: steps to distinguishing type of sequence
- classwork / exit slip: Sequences Match
- spend rest of class working on project
- homework: finish project due FRIDAY
- extra credit homework: Word Search

FRIDAY:
- collect classwork/homework packets (for stamps)
- presentations

Warm-up

Brief overview of what it means to be a sequence, basic examples of sequences, Can you guess what the next number/shape/picture would be?

Sequences in nature / other cool real world stuff

Instruction

Give a sequence and ask students to find the 100th number/shape/picture in the sequence (at this point the lesson can go in a few directions...hence diversity).

Work with finding the 100th term, pose less challenging sequence-index, or move on...
